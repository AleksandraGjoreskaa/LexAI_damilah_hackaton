# 🏛️ LexAI — Македонски AI Правен Асистент

> Интелигентен асистент кој одговара на правни прашања базирано на македонското законодавство, користејќи RAG (Retrieval-Augmented Generation) технологија.

---

## 🎯 Проблем

Македонските граѓани и правници немаат лесен начин да пребаруваат и добијат одговори од правни текстови. Законите се расфрлани по PDF документи, тешки за навигација, и бараат многу време за ручно пребарување.

## 💡 Решение

**LexAI** е AI правен асистент кој:
- Автоматски ги вчитува законите од Министерство за правда
- Го разбира прашањето на корисникот на македонски јазик  
- Пребарува низ 3,601 правен сегмент со семантичко пребарување
- Генерира прецизен одговор цитирајќи конкретни членови од законот

---

## 🖥️ Демо (Скриншот опис)

### Разговор (Chat)
- Поставете прашање на македонски → Добивате одговор со извори
- Стриминг одговор во реално време (token-by-token)
- Markdown форматирање (bold, листи, наслови)
- Колабирачки извори со процент на релевантност
- Follow-up прашања генерирани од AI
- Копирај одговор / Преземи PDF на секој одговор
- Темна тема (toggle во sidebar)

### Пребарување (Search)  
- Директно семантичко пребарување во законската база
- Top-10 резултати со highlight на клучни зборови
- Релевантност оценка за секој резултат

### Документи (Documents)
- Статистика: 31 документ, 3,601 сегмент, 1,800+ страници
- Upload нови PDF закони (drag & drop)
- Автоматска обработка (извлекување → сегментирање → embedding)

## 🚀 Клучни карактеристики

1. **RAG Pipeline** — Retrieval-Augmented Generation за прецизни правни одговори
2. **Streaming** — Одговорот се прикажува token-by-token (како ChatGPT)
3. **Семантичко пребарување** — Наоѓа релевантен текст по значење, не само по клучни зборови
4. **Автоматско скрејпирање** — Вчитува закони директно од Министерство за правда
5. **Квалитетна филтрација** — Автоматски отстранува нечитлив текст од лоши PDF-ови
6. **Мобилен дизајн** — Responsive UI за десктоп и мобилни уреди
7. **Предлог прашања** — По секој одговор, AI генерира 3 follow-up прашања
8. **Споделување/Извоз** — Копирај одговор во clipboard или преземи како PDF документ
9. **Темна тема** — Dark mode toggle со зачувување на преференца

## 🛠️ Технолошки стек

| Слој | Технологија |
|------|-------------|
| Frontend | React 18, Vite 5, TypeScript, Tailwind CSS 3 (dark mode) |
| Backend | Python 3.13, FastAPI, Uvicorn |
| Vector DB | ChromaDB (локален, persistent) |
| Embeddings | intfloat/multilingual-e5-base (HuggingFace) |
| LLM | GPT-4o (преку GitHub Models) |
| PDF обработка | pdfplumber + LangChain text splitters |
| Web scraping | BeautifulSoup4 + requests |
| Database | SQLite (async, SQLAlchemy 2.0) |
